D01-Anthony Di Fatta's Design Principles

Essential Principles for Design

  • Visibility
  • Consistancy
  • Feedback
  • Constraints
  • Affordances
  • Accessibility

Considerations for Designing the Conceptional Model

  • Design the conceptional model purposely don't let it arise from the technology
  • Be wary of a poor or erroneous model
  • Make sure the mental model and the conceptional model match

Considerations for Simplicity, Assessibility, and Cultural Inclusion

  • Use icons
  • Make a site simple to navigate
  • Think about the way the website is organized
  • Learn by doing
  • Easy to understand features quickly
  • Be able to remember the features
  • Be as easy as email
  • Use multiple visual cues
  • Have clear instructions
  • For text on sites, include text to speech, and other similar options
  • Use alternative text on images and other graphics to make them stand out, and easier to read
  • Think about the language and symbols used
  • Do not assume two places close have same culture
  • Allow for different languages to be use on a site
  • Be aware of the difference in direction that people could read a site

Human and Emotion Design Principles

  • Provide discoverability
  • Provide feedback
  • Utilize a conceptional model
  • Ensure you have good mapping
  • Include affordances, signifiers, constraints
  • Be wary of colors that are used
  • Think about the age demographic
  • Think about the shapes of icons
  • Have empathy for your users

Web-Site Interface Principles

  • Take advantage of conventions
  • Create effective visual hierarchies
  • Break pages up into clearly defined areas
  • Make it obvious what's clickable
  • Eliminate distractions
  • Format content to support scanning
  • Provide breadcrumbs

Mobile Devices Principles

  • Make sure everything is visable
  • Have easy scanning for the users
  • Should allow zooming
  • Have a cursor
  • Have strong affordances for the smaller display
  • Have a design that is aesthetically pleasing, and highly visable
  • Provide easy access to common tasks
  • Provide real-time feedback
